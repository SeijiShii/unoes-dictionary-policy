# 📄 UNOES Dictionary プライバシーポリシー

このプライバシーポリシーは、UNOES Dictionary（以下、「本アプリ」または「当社」）が、本アプリを通じて収集するユーザー情報とその取扱いについて説明するものです。

**最終更新日: 2025-12-03**

### 1. 収集する情報とその利用目的

本アプリは、サービスの提供、改善、広告配信、およびセキュリティの確保のために、以下の情報を収集し、利用します。

| 収集する情報の種類 | 収集方法 | 利用目的 |
| :--- | :--- | :--- |
| **アカウント情報** | ユーザーによるログイン時 | サービスの利用者の特定、ログイン機能の提供、サブスクリプション状況の管理、データの同期。 |
| **個人識別情報** | Google/メールアドレスでのログイン時 | **Googleアカウント**: Google User ID、氏名、メールアドレス（プロフィール画像は現在収集していません）。**Emailログイン**: メールアドレス、パスワードハッシュ。 |
| **認証情報** | ログイン時 | JWTトークン、IDトークン（Google認証時）、クライアントフィンガープリント。認証状態の管理、セキュリティの確保。 |
| **サブスクリプション情報** | サブスクリプション購入時 | 購入トークン（Purchase Token）、プランID、サブスクリプションステータス、開始日時、有効期限、キャンセル日時。サブスクリプションの管理、アクセス権の付与。 |
| **決済情報** | サブスクリプション購入時 | ユーザーのサブスクリプションプランの購入・管理、サービスへのアクセス権付与。（※決済処理は**Google Play Store**によって行われ、当社がクレジットカード情報などの機密情報を直接保持することはありません。） |
| **広告識別子** | 広告表示時 | Google AdMobによる広告識別子（広告ID）。ユーザーの興味関心に合わせた広告の表示、広告効果の測定。 |
| **利用状況に関する情報** | アプリ利用時 | APIリクエストログ（リクエストパス、メソッド、ステータスコード、レスポンス時間）、クラッシュレポート、パフォーマンスデータ。サービスの改善、新機能の開発、障害対応。※**辞書検索履歴は保存されていません。**検索はリアルタイムで処理され、検索内容はサーバーに保存されません。 |
| **デバイス情報** | アプリ利用時 | デバイスの種類、OSバージョン、言語設定、プラットフォーム情報（Android/Web）、クライアントIPアドレス、User-Agent。サービスの安定性の確保、障害対応、ユーザーサポート、セキュリティの確保。 |
| **ログ情報** | APIリクエスト時 | リクエストメソッド、パス、クエリパラメータ、ステータスコード、レスポンス時間、クライアントIPアドレス、User-Agent、リクエスト/レスポンスヘッダー（環境変数で制御可能）。サービスの運用、セキュリティ監視、障害調査。 |

### 2. サブスクリプションと決済情報

本アプリはサブスクリプションプランを提供しており、収益化に利用しています。

* ユーザーがサブスクリプションを購入する際、実際の決済処理はGoogle Play Storeを通じて行われます。
* 当社は、お客様の**クレジットカード番号や銀行口座情報などの機密性の高い決済情報を直接収集・保持することはありません。**当社が受け取るのは、取引のステータス、購入した商品ID、有効期限など、サブスクリプションを管理するために必要な情報のみです。

### 3. 広告について

本アプリは、広告収入を得るために第三者配信事業者（**Google AdMob**）による広告表示を実装しています。

* 本アプリでは、以下の種類の広告を表示しています：
  * **バナー広告**: アプリ画面下部に常時表示される広告
  * **インタースティシャル広告**: 画面遷移時に表示される全画面広告
  * **リワード広告**: 視聴完了により24時間広告非表示となる報酬型広告
* 広告配信事業者（Google AdMob）は、ユーザーの興味関心に基づいた広告を表示するため、**広告識別子（広告ID）**やCookie、その他の技術を使用して情報を収集・利用する場合があります。
* これらの情報収集および利用は、Google AdMobのプライバシーポリシー（https://policies.google.com/privacy）に従って行われます。
* ユーザーは、デバイスの設定から広告識別子をリセットまたは無効化することができます。

### 4. 情報の第三者提供

本アプリは、以下の場合を除き、ユーザーの個人識別情報を第三者に提供することはありません。

1.  法令に基づく場合。
2.  サービス維持に必要な業務委託先（データ分析、サーバーホスティング、クラッシュレポート、広告配信）に対し、秘密保持契約を締結した上で、必要な範囲内で提供する場合。
3.  ユーザーが明示的に同意した場合。

**第三者サービス提供者:**
* **Google AdMob**: 広告配信サービス（広告識別子、デバイス情報等）
* **Google Play Store**: サブスクリプション決済処理（決済情報はGoogle Play Storeが直接処理）
* **Google Sign-In**: 認証サービス（Googleアカウント情報）

### 5. 情報の管理とセキュリティ

当社は、収集した情報の漏洩、紛失、破壊、改ざんを防止するため、適切なセキュリティ対策を講じ、情報を厳重に管理します。

**セキュリティ対策:**
* ユーザーのパスワードは**ハッシュ化**して保存し、復元できない仕組みを導入しています。
* 認証には**JWT（JSON Web Token）**を使用し、トークンの有効期限を設定しています。
* API通信は**HTTPS**を使用して暗号化されています。
* ログ情報は環境変数で制御可能であり、機密情報（パスワード、トークン等）はログに記録されないよう配慮しています。
* データベースへのアクセスは適切に制限されています。

### 6. ユーザーの権利

ユーザーは、当社が保有する自身の情報について、以下の権利を有します：

* **開示請求**: 当社が保有する個人情報の開示を求めることができます。
* **訂正請求**: 誤った個人情報の訂正を求めることができます。
* **削除請求**: 個人情報の削除を求めることができます。
* **利用停止請求**: 個人情報の利用停止を求めることができます。
* **データポータビリティ**: 個人情報の提供を求めることができます。

ご要望は、本ポリシー末尾に記載の連絡先までご連絡ください。合理的な期間内に対応いたします。

**注意事項:**
* アカウント削除により、サブスクリプション情報も削除される場合があります。サブスクリプションの返金については、Google Play Storeのポリシーに従います。
* ログ情報は一定期間保存され、セキュリティ監視や障害調査に使用される場合があります。

### 7. データの保存期間

当社は、以下の期間、個人情報を保存します：

* **アカウント情報**: アカウントが削除されるまで、またはユーザーから削除請求があった場合
* **サブスクリプション情報**: サブスクリプションが有効な期間、および法的要件に基づく保存期間
* **ログ情報**: セキュリティ監視および障害調査のために必要な期間（通常は90日以内）
* **認証トークン**: トークンの有効期限まで（通常は数時間から数日）

### 8. 本ポリシーの変更

当社は、法令の変更やサービスの変更に伴い、本ポリシーを改定することがあります。重要な変更がある場合は、アプリ内または本アプリのウェブサイト上でお知らせします。変更後も本アプリを継続してご利用いただく場合、変更後のプライバシーポリシーに同意したものとみなされます。
